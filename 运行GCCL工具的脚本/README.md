

# 脚本使用说明

这里提供`bat`和`sh`两种脚本，注意这里只是将脚本进行备份，使用时需要将jar和脚本放在同一文件夹下，可直接下载最新发布包：[![GitHub release (latest by date including pre-releases)](https://img.shields.io/github/v/release/yansheng836/GCCL?include_prereleases)](https://github.com/yansheng836/GCCL/releases)。如果仅仅是版本更新，脚本没有修改将不更新该文件，请以最新发布包为准。

## `运行GCCL工具的脚本.bat`

bat为Windows下的批处理命令，直接双击即可运行。

## `运行GCCL工具的脚本.sh`

sh为Unix或者是类Unix（Linux）下使用的脚本，可在Linux下使用，也可在Windows模拟Linux的环境下使用，如Git（注意：可能会出现控制台乱码的情况，和git的控制台使用的编码有关）。运行的命令如下：

```shell
sh ./运行GCCL工具的脚本.sh
```
